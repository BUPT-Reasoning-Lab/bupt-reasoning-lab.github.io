# 图片资源说明

此目录用于存放实验室主页所需的图片资源。

## 需要的图片

### 1. Favicon（网站图标）
- **文件名**: `favicon.png`
- **尺寸**: 32x32 或 64x64 像素
- **格式**: PNG（支持透明背景）
- **用途**: 浏览器标签页图标

### 2. 实验室 Banner（社交媒体分享图）
- **文件名**: `lab-banner.png`
- **尺寸**: 1200x630 像素（推荐）
- **格式**: PNG 或 JPG
- **用途**: 社交媒体分享时显示的预览图

### 3. 实验室 Logo（可选）
- **文件名**: `logo.png`
- **尺寸**: 建议 200x200 像素
- **格式**: PNG（支持透明背景）
- **用途**: 可在导航栏或 Hero 区域使用

## 图片优化建议

1. **压缩图片**：使用工具如 TinyPNG 压缩图片，减小文件大小
2. **WebP 格式**：考虑使用 WebP 格式以获得更好的压缩比
3. **响应式图片**：为不同设备准备不同尺寸的图片

## 占位图片

如果暂时没有图片，可以使用在线占位图服务：
- [Placeholder.com](https://placeholder.com/)
- [Unsplash Source](https://source.unsplash.com/)
